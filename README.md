# Unix

Unix/Linux 命令、Shell 脚本、终端工具和常用系统配置笔记。

## 快速导航

| 目录 | 内容 |
| --- | --- |
| [disk/](disk/) | 磁盘与文件系统工具 |
| [file/](file/) | 文件查找、编辑、链接、压缩与回收站 |
| [media/](media/) | 浏览器、字典、图片、PDF、视频相关命令 |
| [net/](net/) | curl、SSH、SSL、rclone、临时服务等网络工具 |
| [shell/](shell/) | Bash/Fish/Zsh 与 Shell 脚本练习 |
| [sys/](sys/) | 系统信息、mise、Windows 版本、swap 配置 |
| [terminal/](terminal/) | tmux 与 zellij 终端复用器 |
| [text/](text/) | awk、grep、sed 文本处理 |
| [user/](user/) | 用户与权限管理 |
| [vim/](vim/) | Vim/SpaceVim 使用笔记 |

## 目录结构

```
├── disk/             # 磁盘与文件系统工具 (df, ncdu, tree, yazi)
├── file/             # 文件操作 (edit, find, link, trash, zip)
├── media/            # 媒体处理 (browser, dict, image, pdf, video)
├── net/              # 网络工具 (curl, e-stat, rclone, server, ssh, ssl, clash)
├── shell/            # Shell 环境与脚本
│   ├── bash/         # Bash 配置与用法
│   ├── fish/         # Fish 配置与用法
│   ├── scripts/      # Shell 脚本示例与语法练习
│   └── zsh/          # Zsh 配置与用法
├── sys/              # 系统信息与系统配置 (info, winver, mise, swap)
├── terminal/         # 终端复用器 (tmux, zellij)
├── text/             # 文本处理 (awk, grep, sed)
├── user/             # 用户与权限管理 (chmod, user)
└── vim/              # Vim 编辑器
```
