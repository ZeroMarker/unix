 work`                 | 推荐起名字                       |
| 重新连接             | `tmux attach` 或 `tmux a`          | 连接最后一个会话                 |
| 连接指定会话         | `tmux attach -t work`              |                                  |
| 列出所有会话         | `tmux ls` 或 `tmux list-sessions`  |                                  |
| 断开当前会话（不关闭）| `Ctrl+b d`                         | 最常用操作！                     |
| 彻底杀死会话         | `tmux kill-session -t work`        | 或在 tmux 里 `:kill-session`     |

### 快速记忆口诀（最常用 8 个）

1. `Ctrl+b d` → 跑路（断开）
2. `Ctrl+b c` → 新标签
3. `Ctrl+b "` → 横切
4. `Ctrl+b %` → 竖切
5. `Ctrl+b x` → 杀窗格
6. `Ctrl+b n/p` → 切标签
7. `Ctrl+b 方向键` → 移动焦点
8. `Ctrl+b z` → 放大看清楚

建议新手先熟练掌握这 8 个，基本就能满足 80% 的日常需求。
